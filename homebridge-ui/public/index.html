Find your FP2

Scanning your local network for Aqara Presence Sensor FP2 devices. The FP2 needs to be on the same Wi-Fi network as Homebridge.

Scanning…

    No FP2s found

    Make sure the FP2 is powered on and on the same network as Homebridge. If you haven't onboarded it yet, run the Aqara Home app's setup flow first to give it Wi-Fi credentials, then remove it from the Apple Home app to release the HomeKit pairing so this plugin can claim it.

    Enter the setup code

    The 8-digit HomeKit setup code is printed on the sticker on the back of the FP2 (look for a code that looks like 1234-5678). It's also visible in the Aqara Home app under the device's settings.

    Any common format works — 1234-5678, 12345678, or 123-45-678 will all be accepted and converted to the HAP standard form. On Next, the wizard pairs with the FP2 to read its sensors — this can take a few seconds.

    Name it for HomeKit

    This is the name you'll see in the Home app. It can be changed there later — alphanumeric, spaces, and apostrophes only.

    Services & names

    These are the sensors found on your FP2. Rename any of them for the Home app, or turn whole groups off. Sensible defaults are filled in.

    Review and save

    This device will be added to your Homebridge config. Finish saves and restarts the bridge to apply the changes; Save & add another saves and returns to the scan so you can set up more FP2s before restarting.
